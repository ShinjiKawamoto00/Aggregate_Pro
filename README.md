## 表集計および転記ツールのご案内

本ツールは、エクセルで作成した複数ファイルや複数シートの表を、単一の表に集計したり、転記したりすることができるアプリケーションです。

### 集計くんPro_01（集計くんPro Ver.0.1.*）

以下のような特徴がありますので、ある程度、ファイル編集ができる方向けのアプリケーションです。詳しくは [集計くんPro_01](https://github.com/ShinjiKawamoto00/Aggregate_Pro/tree/main/%E9%9B%86%E8%A8%88%E3%81%8F%E3%82%93Pro_01) をご覧ください。

- 集計処理や転記処理は JSON ファイルで指定します
- 集計処理や転記処理は、バッチ処理で実施され、結果の正誤は出力されたエクセルファイルを確認する、またはログファイルを確認するという方法になります
- Windows で Excel ライセンスをお持ちの PC でのみ稼働します
- 1か月の試用期間が付属しています

### 集計くんPro_10（集計くんPro Ver.1.0.*）: 2026年2月 提供予定

以下のように、どなたでも簡単に操作することができます。

- ブラウザでファイルの選択や処理の指定ができるノーコードタイプのアプリケーションです
- 個別の処理結果は画面で直ちに確認できますので、処理の修正が楽に実施できます
- Windows でも Linux でも稼働し、 Excel ライセンスも不要です（CSVにも対応します）
- 1か月の試用期間が付属しています
